# Hotel Booking Analytics Pipeline

An end-to-end Data Engineering project that demonstrates data ingestion, storage, optimization, and analytics using the Hadoop ecosystem.

## Tech Stack

- MySQL
- Sqoop
- HDFS
- Hive
- Hadoop

## Project Flow

Hotel Booking Dataset
        ↓
      MySQL
        ↓
      Sqoop
        ↓
      HDFS
        ↓
 Hive External Table
        ↓
   Partitioning
        ↓
    Bucketing
        ↓
  Hive Analytics
